# BPM Calculator
두 개의 키를 지정하여 얼마나 빠르게 키를 누를 수 있는지 측정할 수 있습니다.

# Feature

## BPM 측정
![bpm_result](./bpm_result_example.gif)

각 버튼을 눌렀을 때의 평균 BPM을 그래프로 확인할 수 있습니다.

## Dark mode
![dark_mode](./dark_mode_example.gif)

좌측 상단의 스위치 버튼을 이용하여 다크 모드를 적용할 수 있습니다.
